## 1. Функціональні вимоги (FR)

| ID | Опис вимоги |
| :--- | :--- |
| **FR-01** | Система повинна дозволяти зареєстрованим користувачам створювати власні вікторини з питаннями та варіантами відповідей. |
| **FR-02** | Система повинна забезпечувати створення ігрових кімнат (сесій) для швидких змагань із друзями за унікальним кодом. |
| **FR-03** | Система повинна підтримувати режим реального часу для надсилання відповідей гравцями. |
| **FR-04** | Система повинна автоматично розраховувати бали на основі правильності та швидкості відповіді. |
| **FR-05** | Система повинна відображати динамічну таблицю лідерів після кожного питання. |
| **FR-06** | Система повинна забезпечувати автентифікацію користувачів через логін та пароль. |
